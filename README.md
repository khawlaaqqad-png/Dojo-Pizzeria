# Pizza Oven

A simple JavaScript function that creates pizza objects with custom crust, sauce, cheese, and toppings.

## How It Works

The `pizzaOven()` function takes 4 parameters and returns a pizza object:

| Parameter | Description |
|-----------|-------------|
| `crustType` | Type of crust (e.g., "deep dish", "hand tossed") |
| `sauceType` | Type of sauce (e.g., "traditional", "marinara") |
| `cheeses` | Array of cheeses (e.g., `["mozzarella"]`) |
| `toppings` | Array of toppings (e.g., `["pepperoni", "sausage"]`) |

## Pizzas Created

| Pizza | Crust | Sauce | Cheeses | Toppings |
|-------|-------|-------|---------|----------|
| Pizza 1 | deep dish | traditional | mozzarella | pepperoni, sausage |
| Pizza 2 | hand tossed | marinara | mozzarella, feta | mushrooms, olives, onions |
| Pizza 3 | stuffed crust | alfredo | parmesan | spinach, garlic, mushrooms |
| Pizza 4 | thin crust | alfredo | cheddar, mozzarella | chicken, red onions |
